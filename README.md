# Веб-страницы - демонстрационные проекты

## Описание
Коллекция веб-страниц, созданных для изучения и демонстрации технологий HTML5 и CSS3.

## Технологии
- **HTML5** - семантическая разметка
- **CSS3** - стилизация, Flexbox, Grid, анимации
- **Google Fonts** - шрифт Roboto

## Структура проекта
```
├── index.html          # Главная страница
├── style.css           # Основные стили
├── assets/             # Ресурсы (изображения, шрифты)
└── Лабораторная.pdf   # Описание выполненной работы
```

## Выполненная работа
1. Создание семантической HTML5 разметки
2. Стилизация с использованием современных CSS3 возможностей
3. Адаптивная верстка
4. Использование Google Fonts для типографики

## Как использовать
1. Откройте `index.html` в браузере
2. Для разработки: отредактируйте HTML и CSS файлы
3. PDF с описанием лабораторной работы находится в корне проекта
