# Портфолио — Рыбаченок Вадим

## Описание
Персональный сайт-портфолио, созданный на HTML5 и CSS3.

## Стек
- **HTML5** — структура страницы
- **CSS3** — стилизация, Flexbox, Grid, адаптивность
- **Google Fonts** — шрифт Roboto

## Структура
```
├── index.html          # Главная страница
├── style.css           # Стили
├── assets/             # Изображения (фото, иконки)
└── Отчет.pdf           # Отчёт по лабораторной работе
```

## Разделы сайта
1. **О себе** — фото и краткая информация
2. **Проекты** — карточки проектов
3. **Навыки** — технические и личные навыки
4. **Контакты** — email, телефон, ссылки на GitHub и Telegram

## Особенности
- Адаптивный дизайн (мобильные устройства)
- Навигация по якорям
- Hover-эффекты на карточках и ссылках
- Flexbox + Grid layout

## Запуск
Откройте `index.html` в браузере:

```bash
# Или через локальный сервер
python -m http.server 8000
```

## Проделанная работа
- Создана структура HTML-страницы с семантическими тегами
- Реализована навигация с якорными ссылками
- Стилизован хедер с тёмной темой
- Создана секция «О себе» с аватаром
- Реализована сетка проектов через CSS Grid
- Добавлены секции навыков и контактов
- Настроена адаптивность через media queries
- Добавлены иконки для социальных ссылок
